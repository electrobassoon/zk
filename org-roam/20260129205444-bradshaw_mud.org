* Environment & Quickstart
  - **Project Root**: `~/bradshawmud`
  - **Activation**: `source venv/bin/activate` (from project root)
  - **Start Command**: `evennia start` (from `~/bradshawmud/bradshawmud`)
  - **Database**: Gramps v6 (JSON data structure)

* Data Schema Reference (Gramps v6)
  - **Core Rule**: Data is stored in nested dictionaries, not lists.
  - **Census Mapping**: Census entries use the **Forms Gramplet**.
  - **Key Structure**:
    #+begin_src python
    # Reminder: event_ref_list is a list of DICTIONARIES
    event_ref_list = [
        {"ref": "handle_id", "role": "Head", "attributes": {...}},
        {"ref": "handle_id", "role": "Wife", "attributes": {...}}
    ]
    #+end_src
